The package containing the Pluggable Look And Feel

It is mandatory to set a Look&Feel before the creation of the first component; the default Look&Feel is the DefaultLookAndFeel which can be set in this way

LookAndFeel.CURRENT = new DefaultLookAndFeel();

The API provides a second Look&Feel called BootstrapLookAndFeel (and using the Bootstrap library) which can be set (for example) in this way

LookAndFeel.CURRENT = BootstrapLookAndFeel.create();
